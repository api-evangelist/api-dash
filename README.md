# API Dash

API Dash is an open source AI-powered cross-platform API client built with Flutter for desktop and mobile, serving as a lightweight alternative to Postman and Insomnia with REST, GraphQL, and WebSocket support.

## APIs

- [API Dash](https://apidash.dev) - Open source cross-platform API client.
